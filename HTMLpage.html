What is HTML?

HTML stands for HyperText Markup Language. HTML is the standard markup language used to create web pages. HTML files are text files made up of various HTML elements that consist of various tags enclosed in angle brackets like <html> or <table>. Web browsers interpret these elements as different objects like lists, links, and pictures and will display them as such. HTML Documents are separated into 3 main types, Structure Elements, <head> and <body>

Sample HTML Code

<!DOCTYPE html>
<html>
<body>
<h1>HTML Example Heading</h1>
<p>HTML Example paragraph.</p>
<ul>
<a href="http://www.google.com">Example Link</a>
<li>List Item 1</li>
<li>List Item 2</li>
<li>List Item 3</li>
</body>
</html>

HTML Elements

HTML elements are the building blocks of web pages. Some elements like <html>, <head>, and <body> can be omitted in HTML5 but it is recommended you always include them. Many elements are able to be nested inside of containing elements.

Each HTML element has different possible attribute fields, some are required and some are optional. It is left to the developer to determine which optional tags they require. As shown in the image to the left, each attribute name requires a corresponding attribute value, typically this value needs to be represented by a String and therefore require surrounding quotation marks.

Common Elements

Tag Description
<html> Declares the file as written in HTML
<head> Defines the page's head, used for page metadata
<title> Defines the page's title
<body> Defines the page's body, containing the pages content
<h1> Defines a header 1 element (can also be h2-h6 which are smaller)
<ul> Defines a unordered (bulleted) list
<ol> Defines a ordered (numbered) list
