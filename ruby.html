Introduction

Ruby is popular general-purpose, object-oriented programming langauge. Although its general-purpose nature allows it to be used for many different things, most developers use it as a server-side programming langauge for desiging web applications.


So how is Ruby different from Java? First off in Java you must compile your code before you can run it. Ruby, However, is known as an interpreted langauge, so everything is determined at runtime, and no compilation is necessary. Also, everything in ruby is an object, even things like integers and characters. Building off of this, there are no static 'types' in ruby like Java has. Rather, everything is dynamically typed, so the type of a variable can change during runtime. Many of the useful and different features Ruby provides will be discussed on this page.

Control Flow in Ruby

Control flow allows for directing what code executes in a program.One can use things like if/else or case statements to direct the flow of a program, similar to that of Java.



In this example, notice how parentheses are not used around the condition of the statements, and neither are brackets. To mark the end of the statement, the 'end' keyword is used. This allows for easier readability of the code.


Conditionals:
The boolean operators and comparisons behave just like java, where (&&,||,!) are for and, or, and not, and (<.<=,>,>=, ==) are just as you'd expect. 'true' is for true and 'false' is for false.

Loops in Ruby

Loops allow you to repeat code over and over, just like you've experienced in Java. Ruby handles these in the same manner, except for loops look a bit different.



While loops in Ruby look almost like while loops in Java, but similar to if/else statements, no parantheses are needed around the condition and the 'end' keyword marks the end of the loop instead of a bracket.


For loops, although behave the same way in Java, are syntactically different. As shown on the right, the statement first consists of the dummy variable, and then is followed by the keyword 'in'. the expression that follows 'in' is what gets iterated over. In this example, 1..4 is the range of numbers 1 through 4. This means that the four loop will iterate four times until it is completed.

One could also iterate over a list in this manner simply by replacing the range (1..4) with the list. This is generally the most common form of iteration used in Ruby, next to the .each function.

While and For loops

while x < 10
puts x
x = x + 1
end

for x in 1..4
puts x
end



The .each function is an easy way to iterate over objects in ruby. In this example, an array is created and is iterated over using the function. The |x| can again be thought of the dummy variable.

.each function

array = [1,2,3,4]
array.each do |x|
puts x
end

Functions

Similar to methods declared in Java, Ruby can declare functions. However declaring functions in Ruby is significantly easier and faster than declaring them in Java.



In the example a function with name hello is created and takes in the variable "name". The name is then printed along with a friendly "Hello" inside the function. In ruby make sure to place the function before any code that will be using it, as seen in the example, otherwise it won't be declared.

Function Example

def hello(name)
  puts "Hello " + name
end

name = "Tony"
hello(name)

Classes

Classes can be used to construct objects. They are relatively easy to make in Ruby, and just like in all of object-oriented programming allow for easier maintainablity of code. We'll explain the basics of making a class here.



To start off, use the keyword 'class', followed by its name. The end of the class definition marked by the 'end' keyword.
---
Next, the constructor must be defined. To do so, make a function named 'initialize'. When an object of the class gets created, the instance variables will be initialized through the function. Note that '@' at the beginning of a variable marks an instance variable. You can throw in all of your class functions inside the class definition as well.
---
Outside of your class definition, you can use the .new function to initialize an object of your class. the .new function will constuct the object using the initialize function inside the class definition.

...
end
